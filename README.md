\# Java Swing Calculator



A basic calculator application built in Java using Swing GUI components.



\## Features

\- Addition, subtraction, multiplication, division

\- Decimal input

\- Clear and delete functions

\- Negative number toggle



\## Built With

\- Java

\- Swing (JFrame, JButton, JTextField)

\- IntelliJ IDEA



\## How to Run

1\. Clone the repository

2\. Open in IntelliJ IDEA

3\. Run `Calculator.java`
